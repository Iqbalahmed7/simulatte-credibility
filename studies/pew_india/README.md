# Study 1B v2 — PEW India Opinion Replication (Persona Generator Rebuild)

## Overview

Study 1B v2 is a complete rebuild of the India opinion replication study using the Simulatte Persona Generator (DEEP tier) in place of the hand-crafted cohort used in v1. The study replicates population-level political, social, economic, and religious opinion distributions among Indian adults across 15 validated questions drawn from four Pew Research Center surveys. Both calibrated and holdout DA exceed the 91% human replication ceiling — the only study in the Simulatte program to date where holdout DA also substantially clears the ceiling.

## WorldviewAnchor Dimensions (India-specific)

The India DEEP tier uses five WorldviewAnchor dimensions calibrated to Indian attitudinal data:

| Dimension | Label | Range |
|---|---|---|
| **IT** | Institutional Trust | 0 = distrust government → 1 = full trust |
| **SCP** | Social Change Pace | 0 = traditional/preservationist → 1 = progressive |
| **CS** | Collectivism Score | 0 = individualist → 1 = collectivist |
| **ESP** | Economic Security Priority | 0 = low concern → 1 = high concern |
| **RS** | Religious Salience | 0 = secular → 1 = highly devout |

## Reproducing a Sprint

```bash
# Prerequisites
pip install anthropic
echo "ANTHROPIC_API_KEY=sk-ant-..." > studies/pew_india/.env

# Dry-run: inspect routing without API calls
python3 studies/pew_india/pipeline/sprint_runner.py --sprint IND-1 --model haiku --dry-run

# Full calibration sprint
python3 studies/pew_india/pipeline/sprint_runner.py --sprint IND-1 --model haiku

# Holdout validation (zero topic anchors)
python3 studies/pew_india/pipeline/holdout_runner.py --run HD-1 --model haiku
```

**Variance protocol:** Submit the same sprint ID with a suffix (`IND-1b`, `IND-1c`). The runner uses the sprint ID as a filename only — routing logic is identical. Target: SD < 2pp across 3 replications.

## Notable Findings

**1. Both calibrated and holdout DA exceed the 91% human ceiling.**
India v2 is the only study in the Simulatte program where holdout DA (95.87%) also substantially beats the ceiling. In every prior study, holdout DA fell well below calibrated DA and, in some cases, below the ceiling itself.

**2. 1.74pp calibration-to-holdout gap — smallest in the program.**
The India political archetypes (BJP-supporter, BJP-lean, neutral, opposition) predict the full range of India opinion dimensions strongly, including religion, gender norms, climate, and democracy attitudes. The generator's IT tier separation across archetypes transfers cleanly to unseen holdout questions.

**3. Ceiling achieved in 1 sprint vs. 22 in v1.**
The Persona Generator's calibrated WorldviewAnchor anchors eliminated the iterative archetype-tuning work that accounted for 21 of v1's 22 sprints and the residual 5.7pp gap below ceiling.

**4. Sarvam enrichment was disabled and not needed.**
The standard WorldviewAnchor pipeline with India political archetypes was sufficient to reach 97.61% calibrated DA. This establishes a baseline for assessing the marginal contribution of Sarvam enrichment in future runs.
